# FurEver-Home-A-Pet-Adoption-Tracking-and-Management-System
Members: Angelica C. Corpuz, Ronin Jacob C. Guevarra, Francine Anne M. Villaraza

FurEver Home is a Python-based Pet Adoption Tracking and Management System that records pet profiles, medical histories, adoption status, and adopter information. It manages adoption applications from submission to approval using an SQLite database and OOP principles.
